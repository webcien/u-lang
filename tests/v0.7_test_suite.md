## Compilation Tests

All examples must:
1. Compile without errors
2. Generate valid C code
3. Link successfully with Zig
4. Produce correct output when executed
5. Generate binaries < 200 KB (statically linked)

## Test Execution

Run all tests:
```bash
cd /home/ubuntu/u-lang
./run_tests.sh
```

Run specific category:
```bash
./run_tests.sh loops
./run_tests.sh conditionals
./run_tests.sh arithmetic
```
